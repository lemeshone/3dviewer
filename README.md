# 3D Viewer

Приложение на C++/Qt для просмотра 3D-моделей в формате Wavefront **`.obj`**.  
Программа парсит вершины и рёбра из файла и отображает каркасную модель в графическом интерфейсе.

---

## Возможности

- Загрузка моделей в формате `.obj`
- Каркасная визуализация (wireframe)
- Аффинные преобразования:
  - перемещение по X/Y/Z
  - масштабирование
  - вращение вокруг осей
- Архитектура MVC:
  - **model** — парсер и хранение данных
  - **View** — Qt GUI, окно + OpenGL-виджет
  - **Controller** — управление связью между моделью и интерфейсом
- Модульные тесты Google Test
- Генерация документации LaTeX

---

## Технологии

- **C++20**
- **Qt 6 Widgets / OpenGL Widgets**
- **qmake6**
- **Google Test**
- **LaTeX** (для документации)
- **Makefile** для сборки и вспомогательных задач

---

## Структура проекта
- Controller/ — контроллер
- View/ — GUI и виджет отрисовки
- model/ — парсер .obj, математика, affine-трансформации
- tests/ — модульные тесты (GTest)
- dvi/ — LaTeX документация
- objModels/ — примеры .obj моделей
- main.cpp — точка входа
- project.pro — Qt project-файл
- Makefile — сборка/тесты/форматирование/дистрибутив

## Сборка и запуск

### Сборка
make install

### Запуск
make run

## Тесты
make test

## Документация
make dvi
